The Code for The One Hundred (TOH)


                    // get values from the page
                    // starts or controller function
                    function getValues() {
                        // get values from the page
                        let startValue = document.getElementById("startValue").value;
                        let endValue = document.getElementById("endValue").value;

                        // we need to validate our input
                        // convert to int
                        startValue = parseInt(startValue);
                        endValue = parseInt(endValue);

                        // check if int
                        if (Number.isInteger(startValue) && Number.isInteger(endValue)) {
                            // call genrateNumbers
                            let numbers = generateNumbers(startValue, endValue);

                            // call displayNumbers
                            displayNumbers(numbers);
                        } 
                        else {
                            alert("You must enter integers");
                        }
                    }


                    // generate from start value to end value
                    // logic function(s)
                    function generateNumbers(sValue, eValue) {

                        let numbers = [];

                        // we wnat to get all numbers from start to end
                        for (let i = sValue; i <= eValue; i++) {
                            numbers.push(i); 
                        }

                        return numbers;
                    }


                    // dsiplay the nubmers and mark even number bold
                    // display or view fucntions
                    function displayNumbers(n) {
                        // to hold html
                        let templateRows = "";

                        for (let i = 0; i < n.length; i++) {
                            let number = n[i];

                            //check if even
                            if (number % 2 == 0)
                            {
                                templateRows += `${number}`;
                            }
                            else
                            {
                                templateRows += `${number}`;
                            }
                            
                        }

                        document.getElementById("results").innerHTML = templateRows;
                    }

The Code is structured with three functions.

getValues

getValues is a fucntion that retrieves the start value and end value for the inputs.

The function then converts (parse) the string values to an integer. Next we check each value to see if it is a number.

If it is a number the application proceeds to call the function generateNumbers by passing the startValue and endValue as parameteres. If it is not a number (NaN) then it shows an alert informing the user that they need to enter integers only.


generateNumbers

generateNumbers is a function that generates an arry of all the numbers between the start value and the end value. It has two parameters that receive and hold the values passed in from the previous function getValues.

It creates a new variable as an array to store the numbers generated. It then uses a for loop to iterate over every number between the start value and the end value, adding each number to the array using push.

It then returns the value of the array back to the calling function.


displayNumbers

displayNumbers is a function used to display the array with even numbers being Bold.

The function receives the array of numbers as a parameter. We then create a string to hold the resulting html code generated for display.

We then use a for loop to iterate over the number array. We check each number to see if it is even or odd, by dividing the number by 2 and checking if the remainder is 0. the is achieved by using the '%' (modis) operand.

If the remainder is 0 the the statement is true and the literal with html code in it is added to the string. We place the bold tag around the number output. If the remainder is 1 the the else staement is run and we add the html code without the bold tags.

Once the for loop completes, we then use document.getelementbyid to find the tag with the id of results. We then place the string we generated into the innerHtml propery.

Note: the striping look is achieved by use of BootStrap classes applied to the table holding the numbers.
